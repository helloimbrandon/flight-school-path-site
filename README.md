# Flight School Path

A simple, mobile-first checklist app to help map a realistic path from flight simulator
experience into real flight training (focused on North Carolina, community-college-first,
scholarship-funded).

Single self-contained `index.html` — no build step. Progress saves to the browser via
localStorage. Open the file locally, or use the live site.

**Live site:** https://helloimbrandon.github.io/flight-school-path-site/

## Features
- Overall + per-section progress bars
- Checklist items with status (Not started / In progress / Done), notes, deadline, and link buttons
- NC aviation school cards, national + NC scholarship cards
- Simulator experience tracker (framed as self-directed study, not FAA flight time)
- Essay builder + recommendation-letter tracker
- Export progress to JSON, print-friendly view, reset with confirmation

> Flight simulator experience can show commitment, consistency, and self-study, but it does
> not replace real-world instruction or FAA flight hours.
